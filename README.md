# POST BLOG (Node.js + Express + PostgresSQL)

Простий додаток для створення/редагування/видалення постів.

## Запуск локально
1. Клонувати репозиторій:
   ```bash
   git clone <your-repo-url>
   cd post-blog

2. Встановити залежності:

    npm install


3. Створити .env в корені (скопіювати з .env.example) і заповнити:

PORT=8080



4. Запустити:

    npm run dev
